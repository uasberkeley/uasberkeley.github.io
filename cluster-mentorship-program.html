What is the Cluster Mentorship Program?

The Cluster Mentorship Program is designed to help cluster members gain knowledge from cluster leaders in the fields of their choice. You can submit a Cluster Interest Form and email Alaina Tripp (alainatripp@berkeley.edu) to be added to a cluster. Every cluster meets around twice a month for activities, presentations, and other programming.

Cluster Competition

Each semester, clusters compete in the Cluster Competition, where the winning cluster earns a prize. Points can be earned by attending club and cluster meetings, participating in cluster games, and academic marathons.
